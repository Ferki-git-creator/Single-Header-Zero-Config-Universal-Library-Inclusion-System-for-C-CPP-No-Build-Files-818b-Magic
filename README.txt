  ╚══════════════════════════════════════════╝

FEATURES:
• Include entire libraries with ONE LINE
• No .a/.so files needed
• Zero configuration
• Cross-platform (Win/Linux/macOS)
• 478 bytes of pure innovation

USAGE:
1. Add lib_sup.h to your project
2. Use magic command:
   #include "lib_sup.h"
   lib_sub_include("/path/to/library/")
3. Compile normally!

EXAMPLE:
// main.c
#include "lib_sup.h"

lib_sub_include("~/dev/SDL2/")  // MAGIC!

int main() {
    SDL_Init(SDL_INIT_VIDEO);
    // Enjoy SDL without .a files!
}
